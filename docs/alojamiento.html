Alojamiento

La página web de Healthy road, estará alojada en Bluehost, un servicio de alojamiento para sitios web de WordPress dirigido a proyectos de tamaño medio [5]. Bluehost ofrece opciones de almacenamiento tanto en hosting compartido como en VPS o servidores dedicados, permitiendo a los usuarios alojar tanto el frontend como el backend de sus aplicaciones en un mismo entorno [6] y todos los planes de alojamiento web de Bluehost ofrecen MySQL [7].

Por otra parte, los dashboards serán generados a través de consultas en MySQL, y usando la API se podrá consumir un archivo JSON en el frontend, que a su vez usará una biblioteca de visualización.
