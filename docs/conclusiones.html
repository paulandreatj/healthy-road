Conclusiones

  • Para entrenar de manera efectiva el modelo de inteligencia artificial, se necesitaban imágenes de muestra que reflejaran con precisión las condiciones de las carreteras en Colombia. Imágenes de carreteras de India, que presentan características similares a las de Colombia, resultaron ser una valiosa fuente de datos. Estas imágenes permitieron al modelo aprender a detectar y clasificar daños de manera precisa, garantizando que el sistema pudiera generalizar adecuadamente y aplicar sus capacidades de reconocimiento a las infraestructuras viales colombianas.
